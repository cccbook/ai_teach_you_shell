# 8. 変数と関数

---

## 8.1 変数の基本

### 基本的な代入

```bash
# 文字列
name="アリス"
greeting="こんにちは、世界！"

# 数値
age=25
count=0

# 空の値
empty=
empty2=""
```

### 変数の読み取り

```bash
echo $name
echo ${name}    # 推奨される形式、より明示的

# ダブルクォート内
echo "私の名前は ${name} です"
```

### よくある間違い

```bash
# 間違い: = の周りにスペース
name = "アリス"   # コマンドとして解釈される

# 間違い: クォートがない
greeting=こんにちは 世界  # "こんにちは"のみ出力される

# 正しい:
name="アリス"
greeting="こんにちは 世界"
```

## 8.2 クォートの技術

### ダブルクォート `"`
変数とコマンド置換を展開する

```bash
name="アリス"
echo "こんにちは、$name"           # こんにちは、アリス
echo "今日は $(date +%Y) 年です"     # 今日は 2026 年です
```

### シングルクォート `'`
そのまま出力、何も展開しない

```bash
name="アリス"
echo 'こんにちは、$name'           # こんにちは、$name
echo '今日は $(date +%Y) 年です'     # 今日は $(date +%Y) 年です
```

### クォートなし
変数がスペースを含まないことが確実な場合のみ使用

## 8.3 特殊変数

```bash
$0          # スクリプト名
$1, $2...   # 位置パラメータ
$#          # 引数の数
$@          # すべての引数（個別）
$*          # すべての引数（1つの文字列として）
$?          # 最後のコマンドの終了コード
$$          # 現在のプロセス PID
$!          # 最後にバックグラウンド起動したプロセスの PID
$-          # 現在のシェルオプション
```

## 8.4 配列

### 基本的な使い方

```bash
# 配列の定義
colors=("赤" "緑" "青")

# 要素の読み取り
echo ${colors[0]}    # 赤
echo ${colors[1]}    # 緑

# すべて読み取り
echo ${colors[@]}    # 赤 緑 青

# 配列の長さ
echo ${#colors[@]}   # 3
```

### 連想配列（Bash 4+）

```bash
declare -A user
user["name"]="アリス"
user["email"]="alice@example.com"

echo ${user["name"]}    # アリス
```

## 8.5 関数の基本

### 関数の定義

```bash
# 方法1: function キーワード
function greet {
    echo "こんにちは！"
}

# 方法2: 直接定義（推奨）
greet() {
    echo "こんにちは！"
}
```

### 関数パラメータ

```bash
greet() {
    echo "こんにちは、$1！"
}

greet "アリス"    # こんにちは、アリス！
```

### 戻り値

```bash
# return: 終了コード用（0-255）
check() {
    if [[ $1 -gt 10 ]]; then
        return 0  # 成功
    else
        return 1  # 失敗
    fi
}

if check 15; then
    echo "10 より大きい"
fi
```

## 8.6 ローカル変数

```bash
counter() {
    local count=0
    ((count++))
    echo $count
}
```

## 8.7 関数ライブラリ

### ライブラリの作成

```bash
cat > lib.sh << 'EOF'
#!/bin/bash

log() {
    echo "[$(date +%H:%M:%S)] $@"
}

error() {
    echo "[$(date +%H:%M:%S)] エラー: $@" >&2
}

confirm() {
    read -p "$1 (y/n) " -n 1 -r
    echo
    [[ $REPLY =~ ^[Yy]$ ]]
}
EOF
```

### ライブラリの使用

```bash
#!/bin/bash

source lib.sh

log "プロセスを開始します"
error "何か問題が発生しました"
confirm "続行しますか？" && echo "続行します"
```

## 8.8 クイックリファレンス

| テーマ | 構文 | 説明 |
|-------|--------|-------------|
| 代入 | `var=value` | = の周りにスペース禁止 |
| 読み取り | `$var` or `${var}` | `${var}` を使用 |
| ダブルクォート | `"..."` | 変数を展開 |
| シングルクォート | `'...'` | 展開なし |
| 引数 | `$1`, `$2`, `$@` | パラメータ取得 |
| 関数 | `name() { }` | 推奨の定義方法 |
| ローカル変数 | `local var=value` | 関数内でのみ |
| 配列 | `arr=(a b c)` | インデックスと連想配列 |
| source | `source file.sh` | ライブラリ読み込み |
